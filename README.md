# The Napoleonic Archive

A Next.js website for fans of the Napoleonic Wars (1789–1815) and Bernard
Cornwell's **Sharpe** novels. Chronicles the battles, the people, the
regiments, the first-hand stories, the fiction and the resources of the age of
powder and empire.

## Stack

- **Next.js 14** (App Router, TypeScript)
- **Tailwind CSS** with a custom dark palette — deep navy, burgundy and gold
- Fonts: *Cinzel* (display), *Cormorant Garamond* (serif body), *Inter* (UI)

## Pages

| Route         | Contents                                                      |
| ------------- | ------------------------------------------------------------- |
| `/`           | Atmospheric landing with feature grid and a Wellington quote  |
| `/battles`    | Marengo through Waterloo, 18 annotated engagements            |
| `/people`     | French, British, Allied and Sharpe-world figures              |
| `/regiments`  | 95th Rifles, Old Guard, Black Watch, K.G.L., etc.             |
| `/stories`    | Memoirs and anecdotes in their writers' own words             |
| `/fiction`    | Sharpe novels in chronological order, with synopses           |
| `/resources`  | Reading lists, museums, archives, maps, podcasts              |

## Running locally

```bash
npm install
npm run dev
```

Open <http://localhost:3000>.

## Build

```bash
npm run build
npm start
```
